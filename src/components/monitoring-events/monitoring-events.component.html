Monitoring and Debugging Event Listeners

Event listeners are a crucial feature while developing User interfaces as they can be very useful to handle, track, detect or debug user interactions performed on the page or on specific elements.

  1. Gain access to the $0 or temp variable which we can then refer to the console.
  2. Use built-in functions in Chrome DevTools console to detect event firings:
    • monitorEvents($0, 'keydown')
    • unmonitorEvents(temp1, ['keydown', 'click'])

Tip: Right click on any DOM element in the Elements tab and
click on Capture Node Screenshot to take a designated screenshot of a DOM element.
